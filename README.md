# Projeto Site de Futebol

## Sobre o Projeto

Este projeto foi desenvolvido como atividade acadêmica para o curso de TI, na matéria de Integração, ensino e comunidade. Tem por objetivo criar a estrutura visual de um site utilizando apenas HTML e CSS.

O tema escolhido foi futebol, permitindo a criação de diferentes páginas dentro do mesmo contexto. O site possui a funcionalidade de navegação entre páginas e a interface desenvolvida em HTML e CSS.

## Tecnologias Utilizadas

- HTML5
- CSS3

## Estrutura do Projeto

O projeto contém as seguintes páginas:

- Landing Page
- Página de Login
- Página de Cadastro
- Dashboard
